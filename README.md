# Proyecto Clase: Material para Alumnos 🚀

Este repositorio contiene el material de apoyo y las actividades prácticas para mis alumnos. Aquí recopilamos los 30 retos que construimos en clase, enfocados en dominar el desarrollo Web nativo.

## La filosofía: Zero Dependencies
Chicos, recordad lo que siempre hablamos: antes de lanzarse a usar librerías externas de 500KB para todo, hay que conocer los fundamentos. El objetivo de este repo es que tengáis a mano ejemplos reales usando exclusivamente el **"Vanilla Stack"**:
*   **HTML5** para que la estructura tenga sentido.
*   **CSS Moderno** (Grid, Flexbox, Variables) para que vuestras interfaces no solo funcionen, sino que se vean profesionales.
*   **JavaScript Puro (ES6+)** para controlar el DOM sin "magia negra" de frameworks.

Aquí no hay dependencias. Si el navegador puede hacerlo de serie, nosotros lo hacemos así.

## Guía de Actividades (01 - 30)

He organizado los retos para que veáis cómo va escalando la dificultad a lo largo del curso:

1.  **Fundamentos UI**: Navbars, Sliders y efectos visuales básicos para empezar a soltar la mano.
2.  **Layouts**: Cómo organizar contenido complejo con Grid y Masonry.
3.  **Interactividad**: Notificaciones, menús complejos y carritos de compra.
4.  **APIs del Navegador**: El uso de modales nativos (`<dialog>`), la potencia de la **Constraint Validation API** para vuestros formularios y Drag & Drop.

## Cómo usar este repo
Cada carpeta es una actividad completa. Entrad, leed los comentarios del código (he intentado explicar el "por qué" de cada decisión técnica) y jugad con los archivos.
